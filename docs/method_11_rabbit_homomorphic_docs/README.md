# ラビット＋準同型マスキング暗号プロセッサ ドキュメント

このディレクトリには、ラビット＋準同型マスキング暗号プロセッサに関する文書が含まれています。

## ディレクトリ構成

- `implementation_plan.md`: 実装計画書
- `requirements.md`: 要求仕様書
- `implementation_plan_chapters/`: 実装計画書を章ごとに分割したファイル
- `requirements_chapters/`: 要求仕様書を章ごとに分割したファイル
- `Makefile`: 分割したファイルを結合するための Makefile スクリプト

## 章の構成

### 実装計画書

実装計画書は以下の 10 の章に分割されています：

1. 開発総責任者プロフィール
2. 実装計画概要
3. ディレクトリ構成と納品物件
4. コンポーネント相関図
5. 処理シーケンス図
6. 実装タスク一覧
7. 実装注意事項
8. 次のステップ
9. 本暗号方式について
10. プロジェクトの求められる品質レベル

### 要求仕様書

要求仕様書は以下の 9 つの主要な章に分割されています：

1. パシ子からの前書き
2. 記入者プロフィール
3. 範囲
4. 真の融合アーキテクチャ
5. 基本要件
6. 攻撃モデルと前提条件
7. 戦略的使用要件
8. 機能要件
9. 統合アーキテクチャと革新的技術

**注意**: 要求仕様書は非常に長いため、一部の章のみを分割しています。完全な要求仕様書は元の `requirements.md` ファイルを参照してください。

## Makefile の使用方法

以下のコマンドを使用して、分割したファイルを結合することができます：

```bash
# すべてのファイルを結合して検証
make

# 実装計画書のみを結合
make implementation

# 要求仕様書のみを結合
make requirements

# 結合結果を検証
make verify

# 生成されたファイルを削除
make clean
```

## ファイル分割について

ファイルの分割は各章のセクションヘッダー（`##` で始まる行）に基づいて行われています。各章は個別のファイルとして保存され、章番号と章名に基づいたファイル名が付けられています。
